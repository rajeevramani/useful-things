## Overview

The Apigee X Integrated Developer Portal allows you to create a customized, self-service experience for developers to discover and consume your APIs. Key features include:

- **Identity Provider Integration**: Support for SAML, OAuth, and other authentication methods
- **API Documentation**: Automated API reference documentation from OpenAPI specifications
- **Developer Onboarding**: Self-service registration and API key management
- **Analytics**: Track API usage and developer engagement

## Getting Started

For information about setting up identity providers for your portal, refer to the [official Apigee documentation](https://cloud.google.com/apigee/docs/api-platform/publish/portal/portal-identity-provider).
